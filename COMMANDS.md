## 🔗 Placeholders for Group Messages

When setting welcome/goodbye messages, use these placeholders:

- `{user}` - Mention the user
- `{group}` - Group name
- `{count}` - Member count

Example:
```
/setwelcome Welcome {user} to {group}! You are member #{count}.
```
